uv package manager

Install using curl:

curl -LsSf https://astral.sh/uv/install.sh | sh

Or.

Install using wget:

wget -qO- https://astral.sh/uv/install.sh | sh

Initilize environment in a desired folder :

uv init --python 3.11.6
uv python pin 3.11.6
uv sync

torch with intel xpu support

Add torch and related packages with xpu support. Similar step to be repeated if you rely on cuda:

uv add torch==2.7.1+xpu torchvision==0.22.1+xpu torchaudio==2.7.1+xpu intel-cmplr-lib-rt intel-cmplr-lib-ur intel-cmplr-lic-rt intel-sycl-rt pytorch-triton-xpu tcmlib umf intel-pti --index-url https://download.pytorch.org/whl/xpu --extra-index-url https://pypi.org/simple --index-strategy unsafe-best-match

Now edit the pyproject.toml file to look as follows: 


[project]
name = "solar-weather"
version = "0.1.0"
description = "Add your description here"
readme = "README.md"
requires-python = ">=3.11.6"
dependencies = [
    "grip>=4.6.2",
    "intel-cmplr-lib-rt>=2025.0.4",
    "intel-cmplr-lib-ur>=2025.0.4",
    "intel-cmplr-lic-rt>=2025.0.4",
    "intel-pti>=0.10.1",
    "intel-sycl-rt>=2025.0.4",
    "jupyter>=1.1.1",
    "jupyterlab>=4.4.10",
    "matplotlib>=3.10.7",
    "numpy>=2.3.4",
    "pandas>=2.3.3",
    "pytorch-triton-xpu>=3.3.1",
    "scipy>=1.16.3",
    "seaborn>=0.13.2",
    "tcmlib>=1.2.0",
    "termcolor>=3.2.0",
    "torch==2.7.1+xpu",
    "torchaudio==2.7.1+xpu",
    "torchvision==0.22.1+xpu",
    "tqdm>=4.67.1",
    "umf>=0.9.1",
]

[tool.uv]
# optional, but recommended for multi-index setups
index-strategy = "unsafe-best-match"  


[[tool.uv.index]]
# Name your indexes
name = "pypi" 
url = "https://pypi.org/simple"
default = true

[[tool.uv.index]]
name = "xpu"
url = "https://download.pytorch.org/whl/xpu"

# Explicitly pin specific packages to the xpu index
[tool.uv.sources]
torch = { index = "xpu" }
torchvision = { index = "xpu" }
torchaudio = { index = "xpu" }
intel-cmplr-lib-rt = { index = "xpu" }
intel-cmplr-lib-ur = { index = "xpu" }
intel-cmplr-lic-rt = { index = "xpu" }
intel-sycl-rt = { index = "xpu" }
pytorch-triton-xpu = { index = "xpu" }
tcmlib = { index = "xpu" }
umf = { index = "xpu" }
intel-pti = { index = "xpu" }

Note how the torch related packages are updated using specific index.

Sync to check compatibility:

uv sync

other packages

Other necessary packages:


# simply add them using uv
uv add numpy pandas scipy matplotlib seaborn termcolor tqdm

# jupyter for editing, can use any other
uv add jupyter jupyterlab

# generate requirements.txt
uv pip freeze > requirements.txt

The rest of the packages are updated using the usual index in the pyproject.toml file while only torch related packages use the defined "xpu" index. This can be repeated similarly for a cuda based setup. The uv add command updates the pyproject.toml accordingly.

executing code

Run jupyter as :

uv run jupyter lab

or a python code as :

uv run python mycode.py

or activate the environemnt by:

source .venv/bin/activate

and deactivate by:

deactivate
